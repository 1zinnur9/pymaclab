Standard RBC Model

This is just a standard RBC model, as you can see.

Model Information

maxlags = 1;

maxleads = 1;

maxinfosets = 0;

maxshocks = 1;

Model Parameters

rho = 0.36;

delta = 0.025;

R_bar = 1.01;

eta = 1.0;

psi = 0.95;

Z_bar = 1.0;

sigma_eps = 0.712;

Nonlinear First-Order Conditions

[1] z(t)*k(t-1)**(rho)+(1-delta)*k(t-1)...

     -k(t)-c(t) = 0;

[2] rho*z(t)*k(t-1)**(rho-1)+(1-delta)-r(t) = 0;

[3] betta*E(t)|c(t+1)**(-eta)*betta*c(t)**...

     (eta)*E(t)|r(t+1)-1 = 0;

[4] z(t)*k(t-1)**(rho)-y(t) = 0;

[5] Z_bar**(1-psi)*z(t-1)**(psi)*eps(t)-z(t) = 0;

Log-Linearized Model Equations

[1] (1-betta*(1-delta))*(1-rho)*k(t-1)=...

     (1-betta*(1-delta))*z(t)-r(t);

[2] (K_bar/C_bar)*k(t)=(K_bar/(betta*C_bar))*k(t-1)...

     +(1+delta*(K_bar/C_bar))*z(t)-c(t);

[3] y(t)=z(t)+rho*k(t-1);

[4] eta*E(t)|c(t+1)=E(t)|r(t+1)+eta*c(t);

[5] z(t)=psi*z(t-1)+eps(t);
